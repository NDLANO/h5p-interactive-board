# NDLA H5P Interactive Board

Combine texts, multimedia and tasks into a presentation.

## Getting started

Install all the dependencies:

```bash
npm install
```

Build project for production:

```bash
npm run build
```

Build continuously

```bash
npm run watch
```

Run tests:

```bash
npm test
```
